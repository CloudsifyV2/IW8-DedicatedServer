# IW8-DedicatedServer

> Community-driven research project focused on creating self-hosted dedicated server and matchmaking infrastructure for IW8-based Warzone deployments.


---

## About

IW8-DedicatedServer is an open-source research initiative aimed at building a dedicated server ecosystem for the IW8 engine.

The project's primary objectives are:

* Dedicated server hosting
* Matchmaking infrastructure
* Lobby management
* Self-hosted backend services
* Online play without LAN restrictions
* Networking and protocol documentation

The repository is currently in the **planning and research phase**.

---

## Planned Features

### Matchmaking
